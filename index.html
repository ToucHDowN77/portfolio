"Kivotos.online()" Immigration Management Information System"

Year: 2021 | Category: database | Status: completed

A standalone module of the main "Kivotos Immigration Information System (2021)" Project, this system is the web implementation of the off-line, 2014, relevant database project mentioned below. It is an online Information System, built in the Wix platform using custom-made Javascript code which aims to manage the field of Immigration Law in Greece. It uses an innovative conceptualization to automate the administrative duties of the professional on Immigration by deploying robust modeling methods and display techniques to group and correlate all dispersed but relevant information regarding every and each type of residence permit in Greece. It is the one and unified source of information for Immigration Law no. 4251/2014 and by converting obscure legalities into plain speaking language terminology, it safeguards successful communication while it promotes a cost-effective, user-friendly and high quality operations' framework for stakeholders.

Immigration Law Digital Library (1849-today)"

Year: 2021 | Category: database | Status: completed

The "Immigration Law Digital Library (1849-2021)", standalone module of the main "Kivotos Immigration Information System (2021)" Project, is a web-based database consisting of 538 documents from 1849 up to today, regarding the management of immigration flows by the greek state the last century. It is a comprehensive and integrated digital repository regarding greek legislation documents on Immigration, consisting of hundrends of various kinds of legal texts.It offers unique and modern research tools for those working in the field by integrating into one place and under one organizational schema all written production on the subject. Written in the Wix platform using custom Javascript code, "Immigration Law Digital Library" is updated every time there is a new document in circulation by the central administration. In terms of efficiency, this system manages to significantly lower Public Service's costs for printing paper, printer ink and hardware wear by canceling the need of upkeeping a printed, traditional-form archive.

Roadmap to immigration in Greece: a Decision-Support System (DSS)"

Year: 2021 | Category: DSS | Status: completed

This standalone module of the main "Kivotos Immigration Information System (2021)" Project is a decision-support system (DSS) which facilitates the selection of the proper type of residence-permit by third-country nationals, through a set of predefined and structured questions, built in the Wix platform using custom Javascript code. Its conceptual basis is the abstract representation of the greek Immigration Law (law no.4251/2014) and the graphical workflow that was initially designed and published in 2014 in PDF format. Currently, the "Roadmap" contains 72 predifined questions branching through a system of 133 nodes of binary "yes/no" questions.

Greek-English Immigration and Administration Glossary"

Year: 2021 | Category: glossary | Status: completed

The "Greek-English Immigration and Administration Glossary" is an online lexicon and one more standalone module of the main "Kivotos Immigration Information System (2021)" Project, providing access to dictionary and semantic information on various key concepts in the field of Immigration and Public Administration. This tool, built on Wix platform using custom-made Javascript code, uses lexical information from eight (8) different dictionaries and glossaries (EMN, IOM, EU, UNHCR, USCIS, etc), providing this way an integrated approach to the semantics of each term, globally. Up to day, 181 terms have been indexed and are available in the tool.

"Kivotos" Immigration Management database

Year: 2014 | Category: database | Status: completed

The "'Kivotos' Immigration Management Database", was the early conception of my current, 2021, web-based "Kivotos" Project. At the end of 2013, when the upcoming Immigration Law no.4251/2014 was introduced to the relevant commitees of the Greek Parliament for discussion, I used MS-Access \to design, programmed and publish a comprehensive Database on Immigration Law and procedures which presented in an innovative and straightforward manner all Greek legislation (a compilation of more than 100 laws, presidential decrees, ministerial decisions, circular directives, etc) on Immigration, using an intuitive yet powerful interface. Through the use of intuitive menus and clickable elements, the user (which was mainly a civil servant), was presented with a wealth of combined information about each type of residence permit, facilitating and optimizing this way the duty of providing high level services to the third-country nationals and their applications for legal status in Greece.

History of Veria (1912-2012): Through the lens of its local newspapers

Year: 2012 | Category: database | Status: on hold

A project on hold, this will be both a local history research and data analytics project about my birth city, Veria, Greece, using as main primary sources the newspapers published the century under research. Besides the purely historical interest, a robust data analysis plan has already been designed and awaits inital launch, which aims to run statistical analyses on the huge corpus of the project, for archival and cataloguing purposes. The project corpus consists of: 20 archival sources (libraries, databases, etc), more than 130 digital books and documents, more than 5.000 digital photographs, more than 45.000 newspaper editions and hundrends of photographs of primary sources from the General Archives of State. Total corpus size is 83GB.
